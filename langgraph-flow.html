כך זורם מידע בגרף

גרף ב־LangGraph הוא מסלול: המידע (ה־State) נכנס בהתחלה, עובר מצומת לצומת, ובכל קשת מותנית הגרף מחליט לאן להתפצל. לחצו על הכפתור — וצפו בזה קורה.

לאן תפנה הקשת המותנית?
קצב:
לחצו על ״הרץ את הגרף״ כדי לראות איך המצב (State) זורם בין הצמתים.
מעוין — כניסה ויציאה (START / END) עיגול — צומת: פונקציה שמעדכנת את המצב מלבן זוהר — קשת מותנית: כאן הגרף מחליט חץ — קשת: המסלול שהמצב עובר בו

המצב State

אובייקט אחד שנוסע בכל הגרף. כל צומת קורא ממנו — וכותב אליו.

{
  "input": "מה מזג האוויר בתל אביב?",
  "route": null,
  "answer": null
}

יומן הריצה 📜

כל צעד בגרף — מוסבר בזמן אמת.

  1. עדיין לא רצנו. לחצו ▶ למעלה.

ארבעה מושגים — וזה כל הסיפור

כל גרף LangGraph, גם המסובך ביותר, בנוי בדיוק מאבני הבניין האלה:

מצב State

הזיכרון המשותף של הריצה: מילון אחד שעובר בין כל הצמתים. מה שצומת אחד כותב — הבא אחריו רואה.

class State(TypedDict): ...

צומת Node

פונקציית Python רגילה: מקבלת את המצב, עושה עבודה (למשל קריאה למודל), ומחזירה עדכון למצב.

graph.add_node("node_1", node_1)

קשת Edge

חיבור קבוע בין שני צמתים: אחרי צומת 3 תמיד מגיע צומת 5. בלי שאלות, בלי תנאים.

graph.add_edge("node_3", "node_5")

קשת מותנית Conditional Edge

פונקציה שמציצה במצב ומחזירה את שם הצומת הבא. זה הרגע שבו הגרף מסתעף — ונהיה חכם.

graph.add_conditional_edges("node_1", route_1)

אותו גרף בדיוק — בקוד

חמש קריאות add_node, כמה קשתות, ו־compile. זה כל מה שצריך כדי לבנות את מה שראיתם למעלה:

from langgraph.graph import StateGraph, START, END

graph = StateGraph(State)

graph.add_node("node_1", node_1)
graph.add_node("node_2", node_2)
graph.add_node("node_3", node_3)
graph.add_node("node_4", node_4)
graph.add_node("node_5", node_5)

graph.add_edge(START, "node_1")                # קשת רגילה — תמיד
graph.add_conditional_edges("node_1", route_1)  # בוחרת: צומת 2, 3 או 4
graph.add_conditional_edges("node_2", route_2)  # מחליטה אם לסיים
graph.add_edge("node_3", "node_5")
graph.add_edge("node_5", END)
graph.add_edge("node_4", END)

app = graph.compile()
result = app.invoke({"input": "מה מזג האוויר בתל אביב?"})
הדגמה אינטראקטיבית של מבנה גרף ב־LangGraph · המבנה זהה לתרשים המקורי: מ־START דרך node_1 והקשת המותנית, בשלושה מסלולים אפשריים, עד END
